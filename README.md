# 🌦️ Weather App

A simple and responsive weather application built using React, Material UI, and OpenWeather API. Users can search for any city and get real-time weather information instantly.

## 🚀 Features

- Search weather by city name
- Real-time weather data using OpenWeather API
- Displays:
  - Temperature
  - Humidity
  - Weather Condition
  - Minimum Temperature
  - Maximum Temperature
  - Feels Like Temperature
- Dynamic weather images based on weather conditions
- Responsive and modern UI using Material UI
- Error handling for invalid city names

## 🛠️ Tech Stack

- React.js
- Material UI (MUI)
- JavaScript (ES6+)
- OpenWeather API
- HTML5
- CSS3
